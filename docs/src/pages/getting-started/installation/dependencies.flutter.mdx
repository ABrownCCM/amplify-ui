Add the `amplify_authenticator` package with [pub](https://pub.dev/):

```shell
flutter pub add amplify_authenticator
```

This will add a dependencies like this to your package's pubspec.yaml

```yaml
dependencies:
  amplify_authenticator: ^0.1.0-rc.5
```

Wrap your app's `MaterialApp` widget with the Authenticator widget, and set MaterialApp's builder to `Authenticator.builder()`:

```dart
@override
Widget build(BuildContext context) {
  // Wrap your MaterialApp app in the Authenticator widget
  return Authenticator(
    child: MaterialApp(
      // Set builder to Authenticator.builder() to protect all routes
      builder: Authenticator.builder(),
      home: const Scaffold(
        body: Center(
          child: Text('You are logged in!'),
        ),
      ),
    ),
  );
}
```
